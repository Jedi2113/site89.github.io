Installation Guide
How to install The Technic Launcher and the Site-89 pack, and allocate RAM.

1. Getting The Technic Launcher

If you already have The Technic Launcher you can skip this step. To download it head to the official page: Technic Launcher — Download.

Download the launcher for your operating system. On Windows it will be TechnicLauncher.exe; on macOS/Linux it may be TechnicLauncher.jar. Once downloaded, run the installer (or double-click the .jar), then sign in with your Microsoft account (Mojang accounts are no longer supported after forced migration).

2. Installing the Pack

After logging in, click Modpacks at the top of the launcher. Use the search bar (top-left) and type "Site-89". Select the Site-89 pack from the results and click Install. Wait for the installation to finish, then press Play to launch the pack.

Tip: If the pack doesn’t appear in search, try refreshing or check that you’re searching the Modpacks list and not the Spotlight/Featured sections.

Adding More RAM

A common error is not allocating enough RAM to Minecraft. If you experience frequent crashes or out-of-memory errors, allocating more RAM typically fixes this. Make sure you have 64-bit Java installed (required to allocate more than ~1GB).

Installing Java (64-bit)

  1. Download a 64-bit Java (OpenJDK builds are recommended) from Adoptium (Temurin) or the official Java page.
  2. Install Java and then restart your computer to ensure it is recognized by the system.

Allocating RAM in Technic Launcher

  1. Open the Technic Launcher.
  2. Click the Launcher Options button at the top (it may also be labeled Launcher Settings).
  3. Open Java Settings or the advanced settings panel.
  4. Find the memory / RAM allocation dropdown (often showing 1 GB by default).
  5. Select 6 GB or 8 GB (8 GB is recommended). Be cautious: allocating too much can impact overall system performance—don’t allocate more RAM than your system can spare.
  6. Save your settings and relaunch the pack.
If you still see performance issues after allocating RAM, check background apps and confirm your system meets the pack’s recommended specs.

Done!

Once installed and properly configured, press Play and connect. If you encounter issues, ask for help on our Discord or consult the Newcomer Guide for additional onboarding steps.
